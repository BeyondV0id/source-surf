# Source Surf

A source discovery and curation platform for finding and exploring high-quality sources.

## Features

- Discover trending sources
- Search and filter repositories
- Curated collections
- Community-driven insights

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```
